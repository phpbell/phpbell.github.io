Odeio CSS

Quando comecei a mexer com HTML por volta do ano 2004 o CSS já era algo complexo, acessível apenas a uma elite de programadores.

As incompatibilidades entre os diferentes browsers tornavam necessária a utilização de programas como o TopStyle que mostravam quais browsers suportavam cada um dos seletores.

O fato é que com o tempo o CSS tornou-se cada vez mais complexo e eu acabei odiando essa linguagem cada vez mais.

Atualmente, com a onda do design responsivo devem exister pelo menos uma meia dúzia de formas diferentes para se construir a estrutura básica (grid) de um site.

Odeio profundamente tudo isso. Sinto como se a internet tivesse involuindo ao adicionar suporte a resoluções de tela tão baixas como a dos smartphones.

CSS-in-JS

Outra onda que surgiu mais recentemente foi a do CSS-in-JS.Provávelmente que promoveu massivamente essa onda foi o React do Facebook.

O React nada mais é do que uma gambiarra criada pelo Facebook para tentar aproximar seus aplicativos mobile do código nativo dos smartphones.

Como efeito colateral disso um bando de developers começaram também a usar essa gambiarra para fazer sites e PWAs. A web nunca esteve tão poluída e pesada por conta desse efeito colateral do React.

De onde veio a idéia que colocar o CSS dentro do JS ou virtualizar o DOM melhoraria a performance do site eu não sei, só sei que browsers mais antigos como o Firefox estão tendo cada vez mais dificuldade de acompanhar todas novas essas gambiarras absurdas que surgem a cada dia.
